# Odin Recipes

This is the first project from **The Odin Project's** Foundations curriculum. The main goal of this project is to demonstrate mastery of HTML fundamentals, including document structure, text tags, image insertion, and navigation through internal and external links.

## Recipes Included
* **World's Best Lasagna:** A classic Italian layered dish with bolognese, béchamel, and pasta.
* **Salmon Teriyaki Bowl:** A balanced Japanese-inspired meal with glazed salmon and fresh greens.
* **Smooth Sweet Tea:** A refreshing Southern classic with a secret trick for a mellow finish.

## Technologies Used
* **HTML5:** Semantic web structure.
* **Git & GitHub:** Version control and deployment.
* **Linux (Ubuntu):** Development environment.
